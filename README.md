# Projeto Yearbook Frontend

Este repositório contém o frontend do projeto Yearbook, uma aplicação desenvolvida para criar e gerenciar anuários digitais escolares. O Yearbook permite que alunos, professores e demais membros da comunidade escolar criem perfis, compartilhem recordações, fotos, mensagens e memórias importantes do ano letivo. O objetivo é proporcionar uma plataforma moderna e interativa para relembrar momentos especiais vivenciados ao longo do período escolar.

## Funcionalidades

- Autenticação de usuários (login e cadastro)
- Criação e edição de perfis estudantis
- Upload de fotos e mensagens
- Visualização de perfis e interações entre membros
- Galeria de fotos do ano letivo
- Área administrativa para gerenciar conteúdo e usuários

## Tecnologias Utilizadas

- Ferramentas de construção de páginas web modernas
- Integração com APIs RESTful para comunicação com o backend
- Soluções para estilização e organização de layouts
- Gerenciamento de rotas para navegação entre páginas
- Ferramentas de build e gerenciamento de dependências como Yarn ou npm

## Como executar localmente

1. Clone este repositório:
   ```bash
   git clone https://github.com/isabeladuetes/projeto-yearbook-frontend.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd projeto-yearbook-frontend
   ```
3. Instale as dependências:
   ```bash
   npm install
   # ou
   yarn install
   ```
4. Inicie a aplicação:
   ```bash
   npm start
   # ou
   yarn start
   ```
5. Acesse `http://localhost:3000` no navegador para visualizar o projeto.
